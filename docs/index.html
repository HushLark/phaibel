Every Request Builds Its Own Software

Other assistants run the same script every time. Phaibel composes a unique program from reusable building blocks — tailored to exactly what you asked for.

1

You speak naturally

"Plan a dinner date with Sarah this Friday on the patio"

2

Phaibel designs a process graph

From a catalog of building blocks — weather, calendar, search, messaging, scheduling — it selects and wires together only the nodes your request needs.

3

The process executes transparently

Each node runs and passes results to the next. You see the full process graph — no black box, no mystery.

Up and Running in 60 Seconds

Install, create a vault, and start chatting with your agent.

$ npm install -g phaibel
$ mkdir my-vault && cd my-vault
$ phaibel init
 
# Add your LLM API key (OpenAI, Anthropic, or both)
$ phaibel config add-provider openai
$ phaibel config add-provider anthropic
 
# Opens http://localhost:3737 — choose a personality and start chatting
Remind me to call the dentist tomorrow
Done! I've created a todo "Call the dentist" due tomorrow morning. I'll remind you at 9 AM.
What's on my plate today?
You've got 3 tasks due and a meeting at 2 PM. Want me to prioritize them?
🔑

Bring Your Own Key. Your API keys from OpenAI or Anthropic stay on your machine. Phaibel never phones home.
